To get dimensions of an array:-
use ndim attribute
a.ndim => 1
b.ndim => 2
to get length of all dimensions
use shape attribute
a.shape => returns a tuple of dimensions
=> (3, )
b.shape => (2, 3) (arrow pointing to 2 is row, arrow pointing to 3 is column)
to get data type of an array:-
a.dtype -> returns string
int 32
every numpy array is int 32 by default
to change dtype of array while creating
np.array([ < > ], dtype = 'int 16') (arrow pointing to L, dtype)
to find size of array in bytes:-
a.itemsize -> size of each element
a.size -> size of the array (no of elements)
a.nbytes -> total bytes (size * itemsize)
to access specific element:-
(Same as list, but in same bracket)
a[r, c, other dimension element index]
(bracket pointing to r, c) same bracket
To get a specific row:-

L-> slicing to get everything default start=0, stop=end
this gets the full column)
to get a specific column:-
a[:, c]

to change a specific element:-
a[1, 5] = num
L-> accessing
L-> assigning

np.arange(start, stop, step, dtype = < >)
it is used to create a 1D array of start, stop and step.
start -> optional -> default=0
step -> optional -> default => 1 (it can also be floating)
stop -> exclusive -> necessary
np.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=< >)
returns an 1D array of evenly spaced numbers over a specified mathematical interval
start: the starting value
end point:
True -> stop is included (default)
False -> stop is excluded
retstep:
True: returns (array, step_size calculated)
L-> in a tuple format
False: only array (Default)
dtype:
the data type of output array
Default = float.
