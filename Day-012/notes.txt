‚ÄčMathematics of numpy
‚Äč-> a + num:-
it will add num to all elements of array a.
‚ÄčSimilarly
‚Äč-> a - num
‚Äč-> a * num
‚Äč-> a / num
‚ÄčWe can also do
‚Äča += 2 => will permanently add 2 to every numbers.
‚ÄčWe can also add both arrays:-
‚Äča + b means a[0] + b[0] , a[1] + b[1] , etc.
‚ÄčIn the new array
[ 1 0 0 1 ] + [ 0 1 1 0 ]
=> [ 1 1 1 1 ]

‚ÄčLinear Algebra:-
‚Äčnp.matmul(a, b) -> returns an array.
performs matrix multiplication on a, b only if the normal matrix multiplication rules are satisfied.
‚Äčnp.linalg.det(a)
finds the determinant of the given matrix.
‚ÄčStatistics:-
‚Äčnp.min(a)
finds the minimum element in the array a
‚Äčnp.max(a)
finds max.
‚Äčaxis = 0 => lowest row
‚Äčaxis = 1 => lowest column
‚Äčnp.sum(stats)
Sum of all elements
‚Äčaxis = 0 => sum of columns
axis = 1 => sum of rows
‚ÄčReOrganizing Arrays:-
‚Äčarr.reshape((r, c))
reshapes the arr to the given size (r, c) and returns the new array
‚Äčnp.vstack((a1, a2)) vertical stack
it stacks a1 on top of a2 and returns the new array. only works when same number of columns.
‚Äčnp.hstack((a1, a2)) horizontal stack
it stacks a1 in front of a2 and returns the new array. only works when same number of rows.
‚ÄčMiscellaneous:-
‚Äčnp.genfromtxt('data.txt', delimiter=',')
it is used to generate arrays from a text document of a specified delimiter
‚Äčarr.astype('type')
returns an array with all values changed to specified type.
‚ÄčBoolean Masking and Advanced Indexing:-
‚Äčif i do a condition of an array,
it returns a boolean Array in which all the elements of True will be True and False will be false.
‚Äčex
a = np.array([1, 2, 5, 6])
a > 3
[False, False, True, True]
‚Äčby using it like a filter, we can do
a[a > 3] => [ 5 6 ]
L> only elements that follow condition
‚ÄčIndexing with list:-
‚Äča = np.array([2, 4, 6, 8, 10, 12])
a[[1, 3, 5]] => [ 4 8 12 ]
L> List of indices of elements we want as new array
L> new array with elements of specified indices
‚Äčnp.any(condition, axis=)
np.all(condition, axis=)
‚Äči) checks for any element satisfying condition
ii) checks for all element satisfying condition
‚Äčaxis = 0 => rows wise
axis = 1 => column wise
