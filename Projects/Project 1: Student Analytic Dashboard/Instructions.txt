Instructions
Prerequisites

Make sure the following are installed before running the project:

Python 3.10 or above
Pandas

Install Pandas using:

pip install pandas

How to Run
1.Clone the repository:
git clone https://github.com/thilak091/Ai-Engineer-journey/blob/3ec567e10bacce33ada850c38e6d49864c08ca8e/Projects/Project 1: Student Analytic Dashboard

2.Open the project folder.
Place the dataset (student.csv) in the same directory as dashboard.py.

3.Run the program.
python dashboard.py

Features:

This dashboard performs:

Dataset inspection
Overall statistics
Department-wise analysis
Top performer identification
Placement analysis
Student risk assessment
Admission trend analysis
Grade generation
CSV export of processed data
