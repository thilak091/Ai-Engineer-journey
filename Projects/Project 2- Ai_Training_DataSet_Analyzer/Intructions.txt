AI Training Dataset Analyzer

A beginner-friendly data analysis project built using NumPy as part of my AI & Data Science learning journey.

This project analyzes training metrics of different AI models and generates a clean, professional report directly in the terminal.

Features
Dataset Information
Overall Training Statistics
Best Performing Model
Fastest Training Model
Lowest Loss Model
Model Ranking by Accuracy
Dataset Health Check
Boolean-Based Analysis
Fine-Tuning Simulation
Export Updated Dataset to CSV
Clean and Structured Terminal Report
Dataset

The project analyzes five AI models using the following metrics:

Number of Training Images
Model Accuracy (%)
Training Time (minutes)
Loss Value

Models included:

CNN
ResNet
MobileNet
EfficientNet
Vision Transformer
Concepts Used

This project demonstrates practical use of NumPy concepts, including:

NumPy Arrays
Indexing & Slicing
Statistical Functions
Boolean Masking
Vectorized Operations
Sorting with argsort()
Aggregation Functions
Matrix Manipulation
Array Updating
Saving Data using np.savetxt()

Sample Output

The program generates a report containing:

Dataset Information

Best Model Analysis

Accuracy Rankings
Dataset Health Report
Boolean Analysis
Fine-Tuning Results
Dataset Export Confirmation
