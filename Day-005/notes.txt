‚Äčdropna() :-
remove rows with not a number (Nan) values
‚Äčsyntax:-
df.dropna(axis=< >, how=< >, subset=[ ])

‚Äčaxis -> 'index' by default
‚Äčhow -> 'any' by default

‚Äčaxis = 'index' / 'columns'

‚Äč-> delete rows
‚Äč-> delete columns

‚Äčhow = 'any' / 'all' -> delete only when all value is missing.

‚Äč-> delete even 1 value is missing

‚Äčsubset = [¬†¬†¬†¬† ]

‚Äč-> name of the columns that only should be checked for nan

‚Äčinplace = to make permanent changes.
‚Äčreplace()
for data cleaning, we can use it to replace unknown/missing string by Nan (nan)
‚Äčex:

‚Äč-> nan value of numpy

‚Äčdf.isna()
returns a data frame of True or False for whether nan is present for all the rows.
‚Äčdf.fillna(val) :- inplace for permanent.
replaces all nan by the given value
ex:-
df.fillna('Missing')
‚Äčdf.dtypes
this attribute returns the data types of the columns
‚Äčint - integers
float - floating value
object - string, series, other objects.
‚ÄčType casting:-
used to change the data type
used function astype()
‚Äčex

‚Äčcaution:-
if nan values are present, we cannot convert to int, it will cause an error. so either fillna(0) then int convert or preferably convert to float.
