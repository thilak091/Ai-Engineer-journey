Index:-
‚ÄčA range of numbers assigned to rows as an identifier of rows by default.
‚Äčset_index(<columnname>, inplace)
used to set a column as the index / label of rows.
‚Äčinplace = True \Rightarrow means changes are done forever. (Permanent.)
‚Äčto reset
‚Äčdf.reset_index(inplace=True)
\rightarrow permanent.
‚Äčto set index column while reading the file itself. use index_col parameter in read_csv()
‚Äčex:
df = pd.read_csv('data.csv', index_col='Reg.No')

‚ÄčUse index when you need a different label and your dataframe has a unique value for each row.


‚Äčsort_index(ascending = True/False)
This method is used to sort the dataframe in index order.
‚ÄčThis also has inplace parameter to make permanent changes.

Filter (filt)
‚ÄčIts a series variable, assigned by us for ease that can be used to filter data from dataframe
‚ÄčCreation:- filt = df['last'] == 'Doe'
‚Äčapplying:-
i) df[filt] \rightarrow returns filtered data
ii) df.loc[filt] \rightarrow same.
‚Äčto use And, Or, Not operator we need to use &, |, ~
‚Äčex:
filt = (df['last'] == 'Doe') | (df['first'] == 'corey')
‚Äčif we need to get the opposite of a filter, we can use ~
df.loc[~filt]
‚Äčisin()
‚Äčto create a filter to allow a range / list of values
‚Äčlet countries = [' ', ' ', ' '] list of countries
filt = df['Country'].isin(countries)
\rightarrow gives only that data whose country is in the list.
‚Äčto apply string filter
‚Äčneed to use str.contains() method
‚Äčex:-
filt = df['Name'].str.contains('Thilak', na=False)
\rightarrow doesn't cause error when value is not string.
