‚Äčto change all column names:- (only all).
df.columns = [<new1>, <new2>, <new3>]
‚Äčto change a speci
‚Äčto apply a set of function to change all column name, use list comprehension
‚Äčex:
df.columns = [x.lower() for x in df.columns]
‚Äčto remove white spaces or other values
df.columns = df.columns.str.replace(' ', '_')
‚Äčto change specific columns:-
df.rename(columns = {<old> : <new>}, inplace)
-> dict of columns
-> to be permanent
‚ÄčTo update a single value of a row:-
‚Äčassign after finding with loc.
‚Äčex:-
df.loc[2, 'Name'] = 'Thilak'
‚Äčfor whole row
df.loc[2] = ['Thilak', 'AIDS', 9.8]
‚Äčto change all values of a column,
use self assignment
ex:-
df['email'] = df['email'].str.lower()
‚Äčapply() -> doesn't change df.
‚ÄčFor series:-
it applies a function to every value
‚Äčex:
df['email'].apply(len)
‚Äčfor data frame:-
it applies the function to each series of a data frame.
‚Äčconsider
df = [ [s1], [s2], [s3] ]
-> data frame made of serieses
‚Äčit applies functions (ex len) to each series (counts elements in each series)
‚Äčif we use axis parameter, then it applies to rows.
‚Äčdf.apply(len, axis = 'columns')
-> applies to rows of values.
‚Äčapplymap()
‚Äčfor a df:-
it applies to every individual value of the df.
‚Äčmap()
‚Äčapplies a dictionary of old: new values to all values of df/series, but it substitutes Nan for unmapped values.
